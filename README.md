# Clasificación Automática de Residuos con CNN 

Proyecto desarrollado para el curso **Principios y Tecnologías de Inteligencia Artificial (PTIA)**.  
Su objetivo es entrenar un modelo de **red neuronal convolucional (CNN)** capaz de clasificar residuos sólidos en diferentes categorías a partir de imágenes.

---

##  Objetivo del Proyecto
Desarrollar una red  CNN que clasifique imágenes de residuos en categorías como:
- **Hazardous**
- **Non-Recyclable**
- **Organic**
- **Recyclable**

---



##  Arquitectura Implementada

El modelo CNN implementado contiene:
- Capas convolucionales
- MaxPooling
- Capas densas totalmente conectadas

Framework:
- **TensorFlow / Keras**

---
